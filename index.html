소개

간단한 자기소개를 적어보세요. 관심 분야, 현재 하는 일, 열심히 배우는 것들 등.

이 페이지는 자기소개 + 링크트리 역할을 겸합니다. 필요하면 버튼을 추가/삭제하세요.

  • 설치 필요 없음 (HTML 단일 파일)
  • 반응형 레이아웃 (모바일/데스크톱 자동 대응)
  • JS 없이도 동작 (FAQ는 <details> 사용)

FAQ

이 페이지를 내 도메인에서 쓰고 싶어요

GitHub Pages에서 커스텀 도메인을 연결하세요. 저장소의 Settings → Pages에서 도메인을 설정합니다.

색상이나 글꼴을 바꾸려면?

:root의 CSS 변수를 수정하면 전체 테마가 바뀝니다. 글꼴은 body{ font-family: ... }를 바꾸세요.

이미지/아이콘은 어떻게?

버튼 안의 이모지(예: 🔗, ✉️)를 다른 이모지로 바꾸거나, 공개 이미지 URL을 <img>로 넣어보세요.
